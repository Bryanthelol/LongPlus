# Long Plus
An addon of WoW for options augmentation (Credits to to Leatrix Plus)

## Features

### Nameplate

1. Show/Hide friendly nameplates.
2. Change the width and height of friendly nameplates.

## How to use

Click the minimap button to open Long Plus.
